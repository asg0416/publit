# New User Onboarding

## Flow

1. User opens 아니근데.
2. User sees the full-screen MapGlot-backed thought map.
3. User grants location permission or chooses quiet browsing fallback.
4. Nearby thoughts appear as moving characters with tag labels.
5. User opens "생각 띄우기."
6. User writes text, chooses or edits tag, selects mood and strength.
7. User creates a thought and sees it join the map.
