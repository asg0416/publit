# New User Onboarding

## Flow

1. User opens Publit.
2. User sees "지금 이 공간의 불꽃" radar.
3. User grants location permission or chooses quiet browsing fallback.
4. Nearby flames appear as simulated particles in the circular radar.
5. User opens "내 불꽃 띄우기."
6. User writes text, chooses or edits tag, selects mood and strength.
7. User creates flame and sees it join the radar.

## Copy

- "같은 공간에 떠 있는 순간의 생각을 불꽃으로 확인해보세요."
- "아직 이 공간에 떠 있는 불꽃이 없어요."
- "첫 불꽃을 띄워볼까요?"
